== Frequently Asked Questions ==

= Can I set a different alert threshold for individual products? =  
Not yet. Currently, the plugin uses a global threshold value. Per-product alert thresholds will be added in a future update.

= Does this plugin work with variable products? =  
Yes, it fully supports both simple and variable products.

= Can I send alerts to more than one email address? =  
At the moment, only one email address is supported. Multiple recipients support is coming soon.
